【内容】
エクセル拡張ツールは、定型業務を支援することを目的としたツールです。
本ツールは以下の機能をアドイン形式で提供します。
  ・シートのメニュー表示                  ブックに含まれるシートをメニューに表示し、選択できます。
  ・シート/ブック比較                     ２つのシート、または２つのブックを比較し差分を一覧出力します。
  ・シート一覧を出力                      ブック内のシートの名称とセル内容を一覧出力します。
  ・シートをソート                        ブック内のシートを名称でソートします。
  ・シートの設定                          ブック内のシートのページ設定などを一括で行います。
  ・文字種の変換                          選択範囲の文字列に、全/半角、大/小文字などの変換を行います。
  ・値のトリム                            選択範囲のセル値に対し、トリム処理を行います。
  ・ハイパーリンク                        セルの入力内容を元にハイパーリンクを一括設定します。
  ・固定長／Markdown表／画像コピー        セルの値を固定長、Markdown表または画像形式でクリップボードにコピーします
  ・グループ処理                          重複データの抽出、グループの自動設定を行います。
  ・日付ピッカー                          日付をマウスクリックで入力できるWindows標準Date Pickerを表示します。
  ・罫線を描画                            表ヘッダおよびデータ部分の罫線を自動的に描画します。入れ子に対応しています。
  ・オブジェクトをセルに合わせる          オブジェクトをセルの境界に合わせて移動・リサイズします。
  ・オブジェクトの補正                    ブロック矢印の先端や角丸四角形の丸みなどの統一、および円弧オブジェクトの角度の設定を行います。
  ・SQL文実行                             ワークシートからのSELECT実行および結果一覧の出力、DMLの実行ができます。
  ・VLOOKUP                               VLOOKUPをマウス操作で設定します。
  ・拡張検索                              正規表現による検索を行います。シェイプ内のテキストなども対象対象です。
  ・ファイル一覧出力                      指定されたフォルダ以下のファイル一覧を作成します。
  ・電話番号補正                          電話番号の市外局番を判定し、適切なハイフンを挿入します。
  ・翻訳                                  選択範囲の文章について日英、英日の翻訳を行います。
  ・日本語表記数値変換                    「25百万」などの日本語での数値表記を数値（25000000）に変換します。
  ・QRコード出力                         セルの値を表すQRコード画像を当該セルに出力します。


【インストール方法】
任意の場所に配置したファイル ExTools.xlam と ExTools_OptPack.xlam のWindowsセキュリティロックを解除したのち、そのまま起動してください。
マクロの警告で「マクロを有効にする」を選択するとリボンに「拡張」タブが現れ、使用できます。
Windowsセキュリティロックの具体的な解除方法や自動起動の方法、翻訳機能を使用する際のアカウント登録については以下のURLを参照ください。
https://koichiro-git.github.io/extools/installation.html


【アンインストール方法】
ファイル ExTools.xlam、ExTools_OptPack.xlam、ExTools_OptPack.ini を削除してください。以上でアンインストールは完了です。
ご自身でクイックアクセスツールバーの設定を行った場合は設定を削除してください。以下のURLの「アンインストール」の項目を参照ください。
https://koichiro-git.github.io/extools/installation.html
